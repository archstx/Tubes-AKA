Proyek ini bertujuan untuk menganalisis dan membandingkan kinerja algoritma pencarian nilai minimum dan maksimum menggunakan dua pendekatan, yaitu iteratif dan rekursif.
Analisis dilakukan berdasarkan waktu eksekusi terhadap berbagai ukuran data (n), serta divisualisasikan dalam bentuk grafik untuk melihat pola performa algoritma.
